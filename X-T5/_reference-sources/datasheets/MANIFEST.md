# Datasheet manifest: every film stock technical sheet known to exist

The complete acquisition map for this archive. The [README](README.md) documents what is already on disk (48 PDFs); this manifest catalogues everything else known to exist, so the archive can grow toward "every datasheet ever published" for the stocks that matter to this bank.

Status legend: **archived** (on disk), **pending** (sheet exists, fetch it with [`fetch-datasheets.sh`](fetch-datasheets.sh) or by hand), **none** (manufacturer never published a full technical sheet; noted so nobody wastes an evening looking).

> Fetched URLs move. Where a direct PDF link is listed it was correct when recorded; when a link dies, the source page listed next to it is the stable place to find the current one. The 2026-07-19 session that wrote this ran in a network-restricted environment, so pending items are researched but not yet downloaded.
