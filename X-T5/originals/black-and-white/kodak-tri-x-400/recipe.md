# Kodak Tri-X 400 — Fujifilm X-T5 (X-Trans V) · B&W · STILLS

The iconic photojournalism B&W: gritty, punchy, deep rich blacks with heavy, distinctive grain and a wide tonal range. Supersedes the earlier reference recipe at `X-T5/reference-recipes/kodak-tri-x-400/` as a research-derived original. Grade rationale in [`knowledge.md`](knowledge.md).

| Setting | Value |
|---|---|
| Film Simulation | Acros + Ye filter (ACROS+Ye) |
| Grain Effect | Strong, Large |
| Dynamic Range | DR200 |
| Highlight (Tone Curve) | +1 |
| Shadow (Tone Curve) | +2 |
| Sharpness | +1 |
| High ISO NR | −4 |
| **Clarity** | **+3** |
| Monochromatic Color | 0 (neutral) |
| ISO | Auto, up to ISO 12800 |
| Exposure Compensation | +1/3 to +1 (typically) |

### Notes
- Clarity +3 is a deliberate exception to the Clarity-0 default: Tri-X's gritty, tactile mid-tone "bite" is a defining, faithful characteristic that only Clarity reproduces. Faithfulness overrides the default here (see [validation-methodology](../../../../Knowledge/validation-methodology.md)).
- Grain Strong, Large = Tri-X's heavy, clumpy grain (vs Acros's fine, HP5's even).
- High ISO by design: Tri-X is often shot/pushed at 800–1600+; higher ISO deepens the gritty look. Let ISO climb.
- Filter: +Ye (natural contrast, slightly darker skies) is the default; +R for a punchier, dark-sky Tri-X.
- Video: grain *and* Clarity are unavailable in movie mode, so video loses most of Tri-X's grit; add grain in post; consider Sharpness +2 and Shadow +2 to partly compensate contrast.
