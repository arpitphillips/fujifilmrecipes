# Kodak Ektachrome 100D (5294) — Fujifilm X-T5 (X-Trans V) · CINEMA · VIDEO

Movie-mode version of the [stills recipe](recipe.md). Movie mode drops Grain, Color Chrome Effect, Color Chrome FX Blue, and Clarity (see [video-mode-settings](../../../../Knowledge/video-mode-settings.md)). Target: standard film-simulation recording, not F-Log/HLG.

| Setting | Value | vs stills |
|---|---|---|
| Film Simulation | Provia / STD | same |
| White Balance | Daylight, +1 Red & −1 Blue | same |
| Dynamic Range | DR200 | same value, set **manually** |
| Tone Curve — Highlight | 0 | same |
| Tone Curve — Shadow | 0 | same |
| Color | **+3** | +1 vs stills, to offset the lost Color Chrome Effect / FX Blue |
| Sharpness | +1 | same |
| High ISO NR | −4 | same |
| Interframe NR | AUTO | movie-only |
| ISO | Auto, up to ISO 800 | same |

- Grain unavailable → this is a fine-grain stock anyway; add subtly in post only if styling toward the heavier "Poor Things variant."
- Clarity was 0 already, nothing lost.
- This is the one cinema recipe built on Provia rather than Eterna, which reflects the reversal-film source; keep that in mind if grading further in post (its native tonal response is firmer/shorter than the Eterna-based negatives).
