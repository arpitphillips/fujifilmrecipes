# Kodak Vision3 50D (5203) — Fujifilm X-T5 (X-Trans V) · CINEMA

Kodak's **finest-grain, cleanest cinema stock** — a low-speed (EI 50), daylight-balanced negative prized by cinematographers for pristine, near-invisible grain and "unrivaled highlight latitude." Sean Baker's *Red Rocket* and countless bright-exterior productions lean on it; Vanessa Whyte BSC has praised the Vision3 daylight stocks for holding 3–4 stops of overexposure outside. This completes the bank's Vision3 family alongside [250D](../kodak-vision3-250d/) and [500T](../kodak-vision3-500t/). Eterna base. Grounded in the Kodak technical data sheet (archived: [`kodak-motion/vision3-50d-5203.pdf`](../../../_reference-sources/datasheets/README.md)) + 2 real scans. See [`knowledge.md`](knowledge.md).

## Stills settings
| Setting | Value |
|---|---|
| Film Simulation | Eterna |
| Grain Effect | Weak, Small |
| Color Chrome Effect | Strong |
| Color Chrome FX Blue | Weak |
| White Balance | Daylight, 0 Red & 0 Blue |
| Dynamic Range | DR400 |
| Highlight (Tone Curve) | −2 |
| Shadow (Tone Curve) | 0 |
| Color | +2 |
| Sharpness | 0 |
| High ISO NR | −4 |
| Clarity | 0 |
| ISO | Auto, up to ISO 1600 |
| Exposure Compensation | 0 to +1/3 (typically) |

This **stills** recipe uses the grain and Color Chrome effects that movie mode can't. For the movie-mode adaptation see [`recipe-video.md`](recipe-video.md).

### Why these values
- **Eterna + DR400 + Highlight −2** — the extended-highlight-latitude core shared with the rest of the Vision3 family (2-stop-plus headroom, per the sheet).
- **Grain Weak, Small** — the sheet's own claim: "Kodak's finest grain [cinema film]." Confirmed in 2 real scans: near-invisible grain in both bright sun and overcast light.
- **WB neutral (0R/0B)** — real scans show accurate, un-shifted colour (natural greens, true sky blue) — this is the *cleanest*, least-stylised member of the family; 250D carries a small warm nudge, 50D doesn't need one.
- **Sharpness 0** (vs 250D's −1) — the datasheet stresses "advances in grain **and sharpness**"; keep it neutral rather than softened to reflect the extra acuity of the slower emulsion.
- **ISO capped at 1600** — the lowest ceiling in the cinema family, reflecting the true EI-50 speed; push it and you lose the entire point of the stock.
- **Clarity 0** — per standard; nothing calls for grit on the cleanest stock in the bank.

### Handling
- **Bright daylight exteriors** — this is what it's built for. Don't use it indoors or at night; reach for [500T](../kodak-vision3-500t/) instead.
- Tolerates generous overexposure (the "3–4 stops over and it still holds" reputation) — lean into +1/3 EC rather than protecting highlights defensively.

**Validation tier:** Datasheet-validated (finest-grain claim, daylight balance, extended highlight latitude — archived PDF) + **characteristic-validated** (Sean Baker's *Red Rocket*, Vanessa Whyte BSC's published praise of the family's highlight latitude) + **2 real scans** (bright sun + overcast, both confirming near-invisible grain and neutral colour). Among the strongest-evidenced originals in the bank.
