# Creative / Editorial Looks

These are **original creative grades** — not film emulations, but designed *looks* inspired by fashion/editorial photography and bold creator styles. "Faithfulness" here means faithfulness to the **aesthetic reference**, not a film datasheet, so the [core-principle](../../../Knowledge/validation-methodology.md) image work is: **drop inspiration images into each look's `references/` folder** (magazine tears, creator posts, mood boards) and I'll tune to them. They're marked as creative designs, not scan-validated films.
