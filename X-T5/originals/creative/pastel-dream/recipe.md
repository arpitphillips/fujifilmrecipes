# Pastel Dream — Fujifilm X-T5 (X-Trans V) · STILLS

A soft, sweet, candy-pastel look, light airy tones, gentle colour that reads sugared rather than saturated, flattering skin. Built on Astia (Fujifilm's soft profile) with a bright, lifted exposure. Creative design; drop pastel mood-boards into [`references/`](references/) to tune. See [creative README](../README.md).

| Setting | Value |
|---|---|
| Film Simulation | ASTIA / Soft |
| Grain Effect | Weak, Small |
| Color Chrome Effect | Off |
| Color Chrome FX Blue | Off |
| White Balance | Daylight, −1 Red & −2 Blue |
| Dynamic Range | DR400 |
| Highlight (Tone Curve) | −2 |
| Shadow (Tone Curve) | −2 |
| Color | +2 |

| High ISO NR | −4 |
| Clarity | −2 |
| ISO | Auto, up to ISO 6400 |
| Exposure Compensation | +2/3 to +1⅓ (typically) — bright is the point |

### Why
- Astia: soft contrast and flattering colour: the pastel-safe base.
- Highlight −2 / Shadow −2 + DR400 + bright exposure: the pastel formula: colour kept (+2) but the *tone* around it lifted and flattened, so hues read as sugared tints.
- WB −1R/−2B: a faint warm-cream lean (minus-blue warms, minus-red stops it going orange) for that milky pastel base.
- Chrome effects Off: depth/richness is the *opposite* of pastel; keep colours light.
- Clarity −2, Sharpness −1: soft, dreamy rendering (justified deviation: softness is the look).

### Variant: "Fruit Pastel" energy
For punchier candy colours, swap the base to Velvia and keep everything else (the community-proven vivid-base/lifted-tone trick): sweeter, louder pastels; slightly riskier skin.
