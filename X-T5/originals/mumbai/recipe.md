# Mumbai (General) — Fujifilm X-T5 (X-Trans V) · STILLS

An original place-look for Mumbai's warm, dry-season, everyday character: the chaotic, textured, sun-drenched street. Earthy rust, brick and terracotta tones, deep dry-season blue skies, and vivid colour-pop accents in the tarps, signage, saris and taxis, with warm skin and a hint of golden haze. This is the warm, lively counterpart to the cool, wet [Mumbai Monsoon](../mumbai-monsoon/recipe.md). Rationale in [`knowledge.md`](knowledge.md).

| Setting | Value |
|---|---|
| Film Simulation | Classic Negative |
| Grain Effect | Strong, Small |
| Color Chrome Effect | Strong |
| Color Chrome FX Blue | Weak |
| White Balance | Daylight, +3 Red & −2 Blue |
| Dynamic Range | DR400 |
| Highlight (Tone Curve) | −1.5 |
| Shadow (Tone Curve) | +1 |
| Color | +3 |
| Sharpness | +1 |
| High ISO NR | −4 |
| Clarity | +1 |
| ISO | Auto, up to ISO 6400 |
| Exposure Compensation | −1/3 to +1/3 (judge per scene) |

### Why these values (short)
- Classic Negative is punchy, characterful, warm-capable and filmic. It makes earthy tones rich and colour accents pop while keeping a nostalgic street feel.
- Daylight +3R/−2B gives a warm golden lean for the Indian sun and haze while the blue sky stays blue.
- DR400 with Highlight −1.5 and Shadow +1 tames Mumbai's harsh-sun contrast and holds the city's ever-present sea-haze glow, those milky dusk horizons, while opening deep alley shadows. (Highlight was refined from −1 to −1.5 after the 11-image soul study; see [`knowledge.md`](knowledge.md).)
- Color +3, CCE Strong and CCFXB Weak give vivid saturated colour, with popping accents and rich rust and terracotta, under a deep dry-season blue sky.
- Grain Strong/Small, Sharpness +1 and Clarity +1 build the gritty, high-texture street. Clarity +1 is a deliberate deviation from the 0 default, because the textured chaos genuinely benefits from a little mid-tone grit. Drop it to 0 for a cleaner look.

### Tuning
- Golden hour and haze: WB +4R/−3B with Shadow 0 runs warmer and glowier.
- Cleaner, less gritty: Clarity 0, Grain Weak/Small.
- Punchier midday: DR200, Color +4.
- For wet, overcast monsoon days, switch to [Mumbai Monsoon](../mumbai-monsoon/recipe.md).
