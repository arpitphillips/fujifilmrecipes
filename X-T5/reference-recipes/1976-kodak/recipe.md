# 1976 Kodak — Fujifilm X-T5 (X-Trans V) · STILLS

Status: Reference capture (attributed), for testing/benchmarking.
Source/credit: [Fuji X Weekly (Ritchie Roesch)](https://fujixweekly.com/2023/08/03/1976-kodak-fujifilm-x-t5-x-trans-v-film-simulation-recipe/), inspired by Joel Meyerowitz's 1970s Kodak work.

A punchy, warm 1970s Nostalgic Neg. look with bright highlights and deep, saturated shadows, a vivid vintage American-color aesthetic (distinct from the softer, faded California Summer despite sharing the base).

| Setting | Value |
|---|---|
| Film Simulation | Nostalgic Neg. |
| Grain Effect | Strong, Small |
| Color Chrome Effect | Strong |
| Color Chrome FX Blue | Off |
| White Balance | Auto, −2 Red & −4 Blue |
| Dynamic Range | DR200 |
| Highlight | +1.5 |
| Shadow | +3 |
| Color | +4 |
| Sharpness | −2 |
| High ISO NR | −4 |
| Clarity | −3 |
| ISO | Auto, up to ISO 6400 |
| Exposure Compensation | +1/3 to +1 (typically) |

### Compatibility
Fifth-generation only (requires Nostalgic Neg.). Note the contrast is *high* here (Highlight +1.5 / Shadow +3, DR200), the opposite approach to California Summer's faded low-contrast take on the same film sim.
