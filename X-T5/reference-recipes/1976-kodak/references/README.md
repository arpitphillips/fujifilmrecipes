# References — drop reference film scans here

Drop **real film scans** of this stock/look here (from Reddit, Flickr, Wikimedia, lab galleries, datasheets, your own reference finds). They are used to **validate and improve** this recipe.

- Analysis-only — these are NOT shipped in the product.
- If you can, note the **source** and **lighting** (sun / overcast / golden hour / indoor) in the filename, e.g. `sunny_street_reddit.jpg`.
- More is better: aim for several across different light conditions so we do not tune to an outlier.
- Reddit auto-scraping is blocked for me, so this folder is the way to feed me good Reddit/Cloudflare-site images.
