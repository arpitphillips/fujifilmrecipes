# Acros Red — High Contrast B&W — Fujifilm X-T5 (X-Trans V) · STILLS

**Status:** Reference capture (attributed) — for testing/benchmarking.
**Source/credit:** [Life, Unintended — Luís Costa](https://lifeunintended.com/articles/my-favorite-fujifilm-film-simulation-settings/) (non-Fuji-X-Weekly source).

A dramatic, high-contrast black & white using **Acros + Red filter** — dark skies, luminous skin/reds, deep blacks. A cleaner (grain-off), higher-contrast alternative to our gritty [Kodak Tri-X 400](../kodak-tri-x-400/recipe.md). Works on all generations that have Acros; direct on X-T5.

| Setting | Value |
|---|---|
| Film Simulation | Acros + Red filter (ACROS+R) |
| Dynamic Range | DR200 |
| Highlight | +3 |
| Shadow | +4 |
| Sharpness | −1 |
| High ISO NR | −4 |
| Grain Effect | Off |
| ISO | ISO 12800 (or Auto-ISO, minimum 2000) |
| Exposure Compensation | +1/3 to +2/3 (typically) |

### Notes
- Very high contrast (Highlight +3 / Shadow +4). The high ISO floor is deliberate — real high-ISO noise stands in for grain (Grain Effect is left Off).
- For a **grainier, mid-contrast** reportage B&W instead, use [Kodak Tri-X 400](../kodak-tri-x-400/recipe.md).
- Add **Monochromatic Color** (warm/cool) if you want a subtly toned rather than pure-neutral B&W.
