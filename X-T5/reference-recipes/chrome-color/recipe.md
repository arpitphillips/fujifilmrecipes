# Chrome Color — a video recipe for Fujifilm cameras (X-Trans V)

Classic Chrome base, designed for movie mode. The documentary entry in Fuji X Weekly's video-recipe series: muted Kodak-leaning colour with a warm shift, made for straight-out-of-camera footage that cuts together like an editorial photo essay.

[Source](https://fujixweekly.com/2026/05/02/chrome-color-a-video-recipe-for-fujifilm-cameras/)

| Setting | Value |
|---|---|
| Film Simulation | Classic Chrome |
| White Balance | Auto, +2 Red & −4 Blue |
| Dynamic Range | DR400 |
| Highlight | 0 |
| Shadow | −1.5 |
| Color | +3 |
| Sharpness | 0 |
| High ISO NR | −4 |
| Interframe NR | Auto |
| ISO | Auto, up to ISO 6400 |

**Notes**
- Shadow −1.5 lifts Classic Chrome's naturally hard shadow floor, which matters more in motion than in stills; crushed shadows read harsher in video.
- The author stresses judging exposure per scene rather than using a fixed compensation, which is good movie-mode discipline generally.
- Compatible with late X-Trans IV bodies (X-T4, X-S10, X-E4, X-T30 II) and everything X-Trans V.
- Compare our [Kodak Gold 200 movie-mode version](../../originals/kodak-gold-200/recipe-video.md) for how a stills-first grade translates the other way.
