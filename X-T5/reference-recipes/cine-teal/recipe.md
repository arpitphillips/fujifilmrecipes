# Cine Teal — Fujifilm X-T5 (X-Trans V, ported from X-Trans IV)

Eterna base. One of the best-known cinematic recipes in the community: a heavy cool white-balance shift and maximum negative Clarity produce a moody teal, movie-frame look that took off on YouTube for both stills and video work. Made for blue hour, shade and overcast light.

[Source](https://fujixweekly.com/2020/06/08/my-fujifilm-x100v-cine-teal-film-simulation-recipe/)

| Setting | Value |
|---|---|
| Film Simulation | Eterna |
| Grain Effect | Weak, Large |
| Color Chrome Effect | Weak |
| Color Chrome FX Blue | Weak *(ported: Strong as published)* |
| White Balance | 4550K, +1 Red & −9 Blue |
| Dynamic Range | DR200 |
| Highlight | −2 |
| Shadow | +4 |
| Color | −1 |
| Sharpness | −2 |
| High ISO NR | −4 |
| Clarity | −5 |
| ISO | Auto, up to ISO 6400 |
| Exposure Compensation | 0 to +2/3 (typically) |

**Notes**
- The −9 Blue shift is the most extreme cool shift in this bank, and it is the entire character of the recipe: everything swims in teal while Eterna keeps contrast soft, except for the Shadow +4 floor that anchors deep blacks under the haze.
- Clarity −5 is the same diffusion trick as [Chrome Fog](../ross-chrome-fog/), used here for cinematic bloom rather than fog. It slows shot-to-shot saving considerably; Fujifilm's workaround is to shoot RAW and reprocess in-camera.
- **X-Trans IV → V port.** Published in 2020 for the X100V/X-Pro3/X-T4 generation with Color Chrome FX Blue Strong. Eterna is a blue-affected simulation, so the X-T5 version drops CCFXB one notch to Weak per the [conversion rule](../../../Knowledge/x-trans-v-and-conversion.md). If your blues look thin, return it to Strong.
- For video the same grade works with Grain, the Chrome effects and Clarity omitted (movie mode lacks them); compensate with Color 0 and Sharpness −3, or see [In the video-recipes wing](../nostalgic-color/) for grades designed video-first.
