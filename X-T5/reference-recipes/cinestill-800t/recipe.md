# CineStill 800T — Fujifilm X-T5 (X-Trans V)

Eterna base. Tungsten-balanced cinematic night-photography look with a blue cast and glow around lights, mimicking the movie-film stock CineStill 800T.

[Source](https://fujixweekly.com/2024/04/16/cinestill-800t-fujifilm-x-trans-v-film-simulation-recipe/)

| Setting | Value |
|---|---|
| Film Simulation | Eterna |
| Dynamic Range | DR400 |
| Grain Effect | Strong, Large |
| Color Chrome Effect | Strong |
| Color Chrome FX Blue | Weak |
| White Balance | Fluorescent 3, -6 Red & -4 Blue |
| Highlight | 0 |
| Shadow | +2 |
| Color | +4 |
| Sharpness | -3 |
| High ISO NR | -4 |
| Clarity | -5 |
| ISO | Auto, up to ISO 6400 |
| Exposure Comp | -1/3 to +2/3 (typically) |

> **Pro tip (from real-frame analysis):** the halation-style glow reads strongest around point light sources against darkness, like neon signs, streetlamps and traffic lights, so compose with practicals in frame; broad even lighting hides the effect. For the clean version of this same emulsion without the glow, use the [Vision3 500T original](../../originals/cinema/kodak-vision3-500t/recipe.md).
