# Fujicolor Super HG v2 — Fujifilm X-T5 (X-Trans V) · STILLS

**Status:** Reference capture (attributed) — for testing/benchmarking.
**Source/credit:** [Fuji X Weekly (Ritchie Roesch, co-created w/ Thomas Schwab)](https://fujixweekly.com/2023/07/20/fujicolor-super-hg-v2-fujifilm-x-t5-x-trans-v-film-simulation-recipe/)

Emulates **Fujicolor Super HG 100/200** consumer negative — a bright, clean, slightly nostalgic everyday Fuji-color look on a Classic Negative base. Notably uses **Auto White Priority** WB and mild positive Clarity (crisper than most).

| Setting | Value |
|---|---|
| Film Simulation | Classic Negative |
| Grain Effect | Weak, Large |
| Color Chrome Effect | Weak |
| Color Chrome FX Blue | Weak |
| White Balance | Auto White Priority, −3 Red & −1 Blue |
| Dynamic Range | DR400 |
| Highlight | −1 |
| Shadow | +1 |
| Color | +2 |
| Sharpness | 0 |
| High ISO NR | −4 |
| Clarity | +2 |
| ISO | Auto, up to ISO 6400 |
| Exposure Compensation | +2/3 to +1 (typically) |

### Sensor-generation translation (from source)
- **X-Trans V (X-T5):** Color Chrome FX Blue **Weak** (as listed).
- **X-Trans IV:** set Color Chrome FX Blue **Strong** (the original IV version used Strong; V renders Classic Neg blues deeper, so it was reduced one notch — see [x-trans-v-and-conversion](../../../Knowledge/x-trans-v-and-conversion.md)).
