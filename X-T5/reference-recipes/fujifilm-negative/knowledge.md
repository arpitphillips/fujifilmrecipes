# Fujifilm Negative — Grade Analysis & Understanding

> Read alongside [`recipe.md`](recipe.md) and the [Knowledge base](../../../Knowledge/README.md).

## What it is
A **clean, neutral modern colour-negative** grade built on the **actual Reala Ace film simulation** — a film sim that exists only on X-Trans V generation bodies (and GFX). Reala Ace combines "faithful reproduction with hard tonality": more colour-accurate than Classic Negative, slightly lower contrast, with notably rich blues. This recipe uses that native profile in a restrained, everyday configuration.

## The grade in one line
A **true-to-life, clean, gently-contrasty colour** — accurate hues, natural skin, fine grain — a dependable modern negative look with no strong stylistic cast.

## How the settings build that grade
| Setting | Value | What it contributes |
|---|---|---|
| **Reala Ace** base | — | Native X-Trans V film sim: faithful colour, hard-ish tonality, rich blues. This is what makes it "real" rather than an approximation. |
| **White Balance** | **5000K, 0R / −2B** | Fixed neutral daylight Kelvin, a hair warm (−2 blue), no red push → clean, repeatable, honest colour. |
| **Dynamic Range** | **DR400** | Soft highlight latitude; ISO floor 500. |
| **Highlight** | −1 | Mildly softened highlights. |
| **Shadow** | −0.5 | Very slightly lifted shadows → gentle, natural contrast. |
| **Color** | +2 | Natural, slightly rich saturation. |
| **Color Chrome Effect** | Strong | Depth in warm/green tones. |
| **Color Chrome FX Blue** | Off | Blues left natural — Reala Ace + X-Trans V already render deep blues, so no extra push needed. |
| **Sharpness** | −1 | Very mildly soft outlines. |
| **Clarity** | −2 | Slightly reduced mid-tone contrast → smooth, easy rendering. |
| **Grain** | Weak, Small | Clean, fine grain. |
| **High ISO NR** | −4 | Preserve detail. |

## Reading the tonal/colour signature
- **Contrast:** low-to-medium and *even* (Highlight −1 / Shadow −0.5). The most "neutral curve" of the colour recipes here.
- **Saturation:** natural (Color +2) with Reala Ace's accurate hue rendering.
- **Cast:** essentially neutral, faintly warm (5000K, −2B) — deliberately un-stylised.
- **Texture:** clean and fine.

## Reala Ace vs Classic Negative here
Both the "Reala Ace" recipe and this "Fujifilm Negative" recipe chase a Reala look, but:
- **Reala Ace** (sibling recipe) approximates it on a **Classic Negative** base with heavy blue emphasis (CCFXB Strong) and a slightly more characterful result.
- **Fujifilm Negative** (this one) uses the **genuine Reala Ace film simulation** for a cleaner, more literally-accurate rendering with natural blues (CCFXB Off).

## Best use
An **everyday true-to-life** grade — documentation, travel, family, general daylight — whenever you want accurate colour with a light filmic touch rather than a strong stylistic statement.

## X-Trans V note
Requires the **Reala Ace film simulation**, which is X-Trans V (5th-gen) only — this recipe **cannot** be used on X-Trans IV or earlier bodies. See [conversion notes](../../../Knowledge/x-trans-v-and-conversion.md).
