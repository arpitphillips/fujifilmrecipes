# Ilford FP4 Plus 125 — Fujifilm X-T5 (X-Trans V)

Monochrome base (deliberately, not Acros). Emulates Ilford FP4 Plus 125 (introduced 1968, emulsion improved 2014), the classic medium-speed B&W stock: "soft and dreamy to sharp and almost graphic," rarely goes fully black, excellent shadow detail. Complements the existing Acros-based B&W originals (Acros 100 / HP5 / Tri-X) with the *fine-art low-speed* pole.

[Source](https://fujixweekly.com/2022/12/28/ilford-fp4-plus-125-fujifilm-x-t5-x-trans-v-x-trans-iv-film-simulation-recipe/)

Notes
- Note the negative exposure-comp convention, slight underexposure protects highlights and builds contrast (unusual in this bank; most recipes run positive).
- Highly tunable per the author: +Ye/+R/+G filters, Clarity -2…+4, and Grain Weak/Small (cleaner) ↔ Strong/Large (grittier).
- WB shift matters even in B&W: it feeds the tonal conversion before desaturation.

> **Datasheet check (2026-07-15):** Ilford's sheet calls FP4 Plus "exceptionally fine grain", Grain Weak/Large is the one grain combination no datasheet supports (Weak/Small would match the spec; the author offers it as an option above). The negative-EC convention matches the sheet's tight EI 50–200 latitude. See [VALIDATION-AUDIT.md](../../VALIDATION-AUDIT.md).
