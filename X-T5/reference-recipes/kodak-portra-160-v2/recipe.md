# Kodak Portra 160 v2 — Fujifilm X-T5 (X-Trans V)

Classic Chrome base. Emulates **Kodak Portra 160** — targeting the merged 2010 Portra 160 (blend of the earlier NC "neutral color" and VC "vivid color" variants). Warm yet natural colour, the low-ISO pole of the Portra family. Completes the bank's Portra trio (160/400/800).

[Source](https://fujixweekly.com/2023/11/10/kodak-portra-160-v2-fujifilm-x-t5-x-trans-v-film-simulation-recipe/)

| Setting | Value |
|---|---|
| Film Simulation | Classic Chrome |
| Grain Effect | Weak, Small |
| Color Chrome Effect | Weak |
| Color Chrome FX Blue | Weak |
| White Balance | Daylight, +4 Red & -5 Blue |
| Dynamic Range | **D-Range Priority Auto** |
| Highlight | — (locked by DR-P) |
| Shadow | — (locked by DR-P) |
| Color | 0 |
| Sharpness | -1 |
| High ISO NR | -4 |
| Clarity | -2 |
| ISO | Auto, up to ISO 6400 |
| Exposure Comp | +1/3 to +1 (typically) |

**Notes**
- Uses **D-Range Priority** instead of a manual tone curve — Highlight/Shadow are unavailable while it's active (see [Knowledge/dynamic-range-and-tone.md](../../../Knowledge/dynamic-range-and-tone.md)).
- Best in sunny conditions (sunrise/sunset through midday); pick another recipe for night.
- X-Trans IV port: raise Color Chrome FX Blue Weak → Strong.
