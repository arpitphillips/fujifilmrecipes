# Kodak T-Max 100 — Hard Tone & Soft Tone — Fujifilm X-T5 (X-Trans V)

Monochrome+G base. Emulates **Kodak T-Max 100** — Kodak's modern tabular-grain B&W: clean, sharp, precise, the tabular-grain counterpart to Ilford's Delta line. Created by Dan Allen after shooting real T-Max 100 rolls side-by-side with an X-T5; because the film shows real frame-to-frame variance (even within one roll), he published **two variants** rather than chase one "perfect" match — pick whichever suits the scene, or split the difference.

| Setting | Hard Tone | Soft Tone |
|---|---|---|
| Film Simulation | Monochrome+G | Monochrome+G |
| Monochromatic Color | WC 0 & MG 0 (Off) | WC 0 & MG 0 (Off) |
| Grain Effect | Weak, Large | Weak, Large |
| Color Chrome Effect | Off | Off |
| Color Chrome FX Blue | Off | Off |
| White Balance | Daylight, −6 Red & −3 Blue | Daylight, −6 Red & −3 Blue |
| Dynamic Range | DR200 | DR200 |
| Highlight | **+0.5** | **−0.5** |
| Shadow | **+2.5** | **+1.5** |
| Sharpness | −1 | −1 |
| High ISO NR | −4 | −4 |
| Clarity | −1 | −1 |
| ISO | Auto, up to ISO 6400 | Auto, up to ISO 6400 |
| Exposure Comp | −2/3 to +2/3 (typically) | −2/3 to +2/3 (typically) |

**Notes**
- The two variants are **identical except Highlight/Shadow**: Hard Tone (+0.5/+2.5) is brighter highlights + deeper shadows = more contrast; Soft Tone (−0.5/+1.5) is softer highlights + gentler lift = less contrast.
- Uses **Monochrome+G** (not Acros) — a cleaner, flatter rendering than the Acros-based recipes elsewhere in this bank, consistent with T-Max's modern, clinical tabular-grain character (vs Acros's richer, more "premium film" gradation).
- Author's own framing: real film has more variance than two presets can capture — "season to taste."
- Compatible with X-Trans IV and V (X-T4, X-S10, X-E4, X-T30 II, X-H2s, X-H2, X-T5, X-S20, X100VI, X-T50).

> **Datasheet check (2026-07-16):** Kodak's T-Max 100 sheet (F-4016, archived: [`kodak-still/t-max-100_f4016.pdf`](../../_reference-sources/datasheets/README.md)) describes T-Max's tabular-grain technology as delivering finer grain and higher sharpness than conventional-grain films at the same speed — consistent with this recipe's Weak grain and negative Sharpness kept modest rather than aggressive (the tabular structure itself does the sharpness work; Fuji's Sharpness setting doesn't need to add more).
