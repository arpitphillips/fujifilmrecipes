# Nostalgia Negative — Fujifilm X-T5 (X-Trans V)

Nostalgic Neg. base. Ritchie Roesch's first recipe made specifically for the X-T5, and still one of the most used. Warm amber highlights, deep shadows and strong small grain give the 1970s American New Color print look the Nostalgic Neg. simulation was built for.

[Source](https://fujixweekly.com/2022/11/22/nostalgia-negative-my-first-fujifilm-x-t5-x-trans-v-film-simulation-recipe/)

| Setting | Value |
|---|---|
| Film Simulation | Nostalgic Neg. |
| Grain Effect | Strong, Small |
| Color Chrome Effect | Strong |
| Color Chrome FX Blue | Weak |
| White Balance | Daylight, +3 Red & −3 Blue |
| Dynamic Range | DR400 |
| Highlight | −1 |
| Shadow | +1 |
| Color | +4 |
| Sharpness | −1 |
| High ISO NR | −4 |
| Clarity | −3 |
| ISO | Auto, up to ISO 6400 |
| Exposure Compensation | 0 to +2/3 (typically) |

**Notes**
- Color +4 on Nostalgic Neg. runs rich; the deep Clarity −3 softens it back toward a printed-photo feel. Clarity at −3 costs save time, so drop to 0 for fast shooting and accept a crisper result.
- Compare [1976 Kodak](../1976-kodak/) and [California Summer](../california-summer/), the other Nostalgic Neg. references in the bank; this one is the warmest and grainiest of the three.
- Native X-Trans V recipe. On an X-Trans IV body the Nostalgic Neg. base doesn't exist, so there is nothing to port backward.
