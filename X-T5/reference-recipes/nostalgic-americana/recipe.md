# Nostalgic Americana — Fujifilm X-T5 (X-Trans V)

Nostalgic Neg. base. A cooler, more faded take on the New Color aesthetic than [Nostalgia Negative](../nostalgia-negative/): Kelvin white balance with a cool shift, muted colour and large grain, closer to an aged print found in a shoebox than a fresh one.

[Source](https://fujixweekly.com/2024/04/29/nostalgic-americana-fujifilm-x-t5-x-trans-v-film-simulation-recipe/)

| Setting | Value |
|---|---|
| Film Simulation | Nostalgic Neg. |
| Grain Effect | Strong, Large |
| Color Chrome Effect | Strong |
| Color Chrome FX Blue | Off |
| White Balance | 5800K, −2 Red & −4 Blue |
| Dynamic Range | DR200 |
| Highlight | −1 |
| Shadow | −1.5 |
| Color | −2 |
| Sharpness | −2 |
| High ISO NR | −4 |
| Clarity | −3 |
| ISO | Auto, up to ISO 6400 |
| Exposure Compensation | judge per scene |

**Notes**
- Where most Nostalgic Neg. recipes push warmth, this one pulls it back with a −2 Red shift on a fixed 5800K base. The amber highlight rendering of the simulation still comes through; it just reads aged rather than golden.
- Strong/Large grain plus Sharpness −2 and Clarity −3 make this one of the softest, most textured recipes in the reference shelf. Clarity −3 slows shot-to-shot saving.
- Native X-Trans V recipe; no conversion needed.
