# Nostalgic Color — a video recipe for Fujifilm cameras (X-Trans V)

Nostalgic Neg. base, designed for movie mode. The recipe that started Fuji X Weekly's video-recipe series: Ritchie Roesch built it for a 72-hour short-film challenge shot on an X-H2s and X-S20, wanting footage that felt like memory. Soft, warm and muted, straight out of camera with no grading.

[Source](https://fujixweekly.com/2026/05/02/nostalgic-color-a-video-recipe-for-fujifilm-cameras/)

| Setting | Value |
|---|---|
| Film Simulation | Nostalgic Neg. |
| White Balance | Auto Ambiance Priority, +3 Red & −3 Blue |
| Dynamic Range | DR400 |
| Highlight | 0 |
| Shadow | −1 |
| Color | −2 |
| Sharpness | 0 |
| High ISO NR | −4 |
| Interframe NR | Auto |
| ISO | Auto, up to ISO 6400 |

**Notes**
- This is a video recipe first. Movie mode has no Grain, Color Chrome effects or Clarity, so the settings list is shorter by design rather than by omission (see [video-mode-settings](../../../Knowledge/video-mode-settings.md)).
- Auto Ambiance Priority is the warmth-keeping AWB mode: it preserves the colour of ambient light instead of correcting it away, which is what makes indoor and golden-hour footage feel remembered rather than documented.
- For stills, the author's own guidance: pick a Grain size (Off or Weak/Small), leave both Chrome effects Off, and keep Clarity at 0.
- Sits alongside [Chrome Color](../chrome-color/), [Vivid Fujichrome](../vivid-fujichrome/) and [Reala Color](../reala-color/) in the same video-recipe series.
