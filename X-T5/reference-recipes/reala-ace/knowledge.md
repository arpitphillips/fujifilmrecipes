# Reala Ace — Grade Analysis & Understanding

> Read alongside [`recipe.md`](recipe.md) and the [Knowledge base](../../../Knowledge/README.md).

## What it emulates
Two layers of meaning:
1. The film: *Fujicolor Reala* (and the Japan-only *Reala Ace*) was Fuji's first "4th-layer-technology" colour negative, a pro/wedding film marketed alongside Superia, with slightly less saturation than Superia but deeper, richer blues, and excellent skin.
2. The film simulation: Fujifilm later introduced a Reala Ace *film simulation* (first on the GFX100 II) described as combining faithful reproduction with hard tonality, essentially a more colour-accurate, slightly lower-contrast, bluer Classic Negative.

This recipe was made before the Reala Ace film sim reached X-series, so it *approximates* that look on a Classic Negative base. (Contrast with the sibling recipe "Fujifilm Negative", which uses the real Reala Ace film sim; see below.)

## The grade in one line
A true-to-life, balanced colour, Classic Negative's character but calmer and more accurate, with a bit more blue and a clean, natural rendering.

## How the settings build that grade
| Setting | Value | What it contributes |
|---|---|---|
| **Classic Negative** base | — | The Superia-derived character, used here in a *restrained* configuration to read as accurate rather than punchy. |
| **White Balance** | **Auto, −1R / +1B** | Auto WB for adaptability, nudged *cooler* (−red, +blue) → the extra blue that distinguishes Reala from Superia. |
| **Dynamic Range** | **DR400** | Soft highlight latitude; ISO floor 500. |
| **Highlight** | −1.5 | Softened highlights, natural top end. |
| **Shadow** | −2 | *Lifted shadows* → lower contrast, "faithful"/true-to-life tonality (not moody). |
| **Color** | +2 | Natural, slightly rich colour. |
| **Color Chrome Effect** | Strong | Depth in warm/green tones. |
| **Color Chrome FX Blue** | **Strong** | Maximum blue depth → the Reala blue signature. (Note: kept Strong here.) |
| **Sharpness** | 0 | Neutral — neither crisp nor soft. |
| **Clarity** | −2 | Gently reduced mid-tone contrast. |
| **Grain** | Weak, Small | Clean, fine grain → a fine pro negative. |
| **High ISO NR** | −4 | Preserve detail. |

## Reading the tonal/colour signature
- Contrast: low and even (Highlight −1.5 / Shadow −2 both soften) → the "faithful", true-to-life curve rather than a stylised split.
- Saturation: natural (Color +2); the Classic Neg base adds character without going loud.
- Cast: cool-neutral with emphasised blues (Auto −1R/+1B + CCFXB Strong).
- Texture: clean and fine (Weak/Small grain, 0 sharpness), one of the tidier, more "modern" looks here.

## Best use
An everyday, all-purpose, true-to-life colour grade, good where you want accurate-but-characterful colour: travel, street, general daylight, and people. Calmer and more neutral than Pacific Blues.

## X-Trans V note
This recipe is written for X-Trans V and uses Color Chrome FX Blue Strong deliberately (the look wants deep blues). Because it's authored natively for V, there's no IV→V blue reduction to apply here. See [conversion notes](../../../Knowledge/x-trans-v-and-conversion.md).
