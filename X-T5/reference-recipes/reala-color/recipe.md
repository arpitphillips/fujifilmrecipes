# Reala Color — a video recipe for Fujifilm cameras (X-Trans V)

Reala Ace base, designed for movie mode. The true-to-life entry in Fuji X Weekly's video-recipe series: clean, balanced colour with a slight cool lean, for footage that should look like unfussy real life rather than a grade.

[Source](https://fujixweekly.com/2026/05/02/reala-color-a-video-recipe-for-fujifilm-cameras/)

| Setting | Value |
|---|---|
| Film Simulation | Reala Ace |
| White Balance | Auto, 0 Red & −3 Blue |
| Dynamic Range | DR400 |
| Highlight | +1 |
| Shadow | −1 |
| Color | +3 |
| Sharpness | −1 |
| High ISO NR | −4 |
| Interframe NR | Auto |
| ISO | Auto, up to ISO 6400 |

**Notes**
- Reala Ace exists as a native simulation on the X-T5 generation, so this needs no approximation; compare the [reala-ace reference](../reala-ace/), which fakes the look on Classic Negative for recipes that predate the simulation.
- Highlight +1 is unusual in this bank and gives video a little sparkle that the flat Reala palette can absorb without clipping, helped by DR400.
- The −3 Blue shift stops AWB from running clinical; it is the one stylistic opinion in an otherwise neutral grade.
- Compatible with X-Trans V bodies that have Reala Ace (X-T5 firmware 4.00+, X100VI, X-T50, X-E5 and later).
