# Reggie's Portra — Fujifilm X-T5 (X-Trans V)

Classic Chrome base, created by Reggie Ballesteros. Currently the single most popular recipe on Fuji X Weekly — versatile portrait look using Auto White Balance (no Clarity adjustment). Originally for X-Trans IV; for X-T5 (X-Trans V) just set Color Chrome FX Blue to **Off** instead of Weak.

[Source](https://fujixweekly.com/2022/06/11/fujifilm-x-trans-iv-film-simulation-recipe-reggies-portra/) / [X-Trans V note](https://fujixweekly.com/2025/12/03/reggies-portra-in-x-trans-v-cameras/)

| Setting | Value |
|---|---|
| Film Simulation | Classic Chrome |
| Dynamic Range | DR-Auto |
| Highlight | -1 |
| Shadow | -1 |
| Color | +2 |
| Noise Reduction | -4 |
| Sharpening | -2 |
| Clarity | 0 |
| Grain Effect | Weak, Small |
| Color Chrome Effect | Strong |
| Color Chrome FX Blue | **Off** (X-T5 adjustment; was Weak on X-Trans IV) |
| White Balance | Auto, +2 Red & -4 Blue |
| ISO | Auto, up to ISO 6400 |
| Exposure Comp | No fixed suggestion — judge each scene |
