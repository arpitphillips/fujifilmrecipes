# Canned Heat (Ross McConaghy) — Fujifilm X-T5 (X-Trans V)

Classic Negative base. A warm, film-like sunset "teal & orange" vibe driven by a massive warm white-balance shift: desaturated overall but with a strong warm cast, deliberately stylised ("100% a vibe"). Created by Ross McConaghy.

[Source](https://www.rossandhisjpegs.com/fujifilm-recipes/canned-heat)

| Setting | Value |
|---|---|
| Film Simulation | Classic Negative |
| Grain Effect | Weak, Large |
| Color Chrome Effect | Weak |
| Color Chrome FX Blue | Weak |
| White Balance | Auto, **+6 Red & −7 Blue** |

| Highlight | −2 |
| Shadow | +1 |

| High ISO NR | −4 |
| Clarity | +5 |
| ISO | Auto (author unspecified) |

Notes
- The +6R/−7B WB is the engine: an extreme warm shift. Combined with Classic Negative's native cool/teal shadows, this creates the teal-orange split (warm highlights/skin, cool shadows) that reads as cinematic.
- Color −2: counter-intuitively desaturated: the warmth comes from the WB, not from saturation, so pulling Color back keeps it from going garish. A different route to teal-orange than our [Kodak 2383 Print](../../originals/cinema/kodak-2383-print/) (which uses positive Color and a milder WB).
- Clarity +5: maximum grit/punch (slows saving).
- Best in sunset / golden-hour light where the warm shift has warm light to amplify.

> **X-Trans V, verified correct as-published (2026-07-16).** Ross authors on an X-Trans V-category body (proven by his Nostalgic Neg. recipe [Pumpkin Soup](../ross-pumpkin-soup/)). Classic Negative is a blue-affected sim, but since the source is already V-generation, Color Chrome FX Blue stays Weak, with no IV→V drop needed for the X-T5.
