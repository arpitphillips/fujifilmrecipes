# Pumpkin Soup (Ross McConaghy) — Fujifilm X-T5 (X-Trans V)

Nostalgic Neg. base. A **warm, punchy, nostalgic autumnal** look — built to "maximise the glorious Autumnal sunlight" with rich golden/amber tones and deep shadows. Best in daylight, golden hour, and sunrise/sunset. Created by **Ross McConaghy**.

[Source](https://www.rossandhisjpegs.com/fujifilm-recipes/pumpkin-soup)

| Setting | Value |
|---|---|
| Film Simulation | Nostalgic Neg. |
| Grain Effect | Weak, Large |
| Color Chrome Effect | Off |
| Color Chrome FX Blue | Weak |
| White Balance | **Underwater, +3 Red & −5 Blue** |
| Dynamic Range | DR200 |
| Highlight | −1 |
| Shadow | **+4** |
| Color | +2 |
| Sharpness | 0 |
| High ISO NR | −3 |
| Clarity | +5 |
| ISO | Auto (author unspecified) |
| Exposure Compensation | author unspecified — daylight / golden hour |

**Notes**
- **Underwater white balance** is the surprise ingredient — an unconventional creative choice (Underwater WB removes blue/adds warmth aggressively), pushed further with +3R/−5B for the deep autumnal amber. A creative-WB trick worth knowing.
- Shadow +4 (maximum) + Clarity +5 (maximum): deep, rich, punchy shadows with heavy mid-tone bite: "warm but punchy," not soft/faded. Distinct from our faded-warm Nostalgic Neg references [California Summer](../california-summer/) and [1976 Kodak](../1976-kodak/).
- Requires the **Nostalgic Neg.** film simulation → **X-Trans V only** (no IV port possible). ✅ Correct for the X-T5 as-published.

> **Sensor-generation verification (2026-07-16):** This recipe is the proof that Ross authors on an **X-Trans V-category body** — Nostalgic Neg. exists only on X-Processor 5 cameras (X-T5, X-H2/S, X-S20, X100VI, X-T50…) and on **no** X-Trans IV camera. That means his whole current recipe set (all six added here) is in the X-Trans V category, i.e. the same blue-rendering generation as the X-T5 — so none of them need the IV→V Color Chrome FX Blue adjustment.
