# Salted Slate (Ross McConaghy) — Fujifilm X-T5 (X-Trans V)

Acros +G base. A **creative dark, slate-grey B&W** with prominent grain — a moody monochrome look named for its slate-like dark grey tonality. Created by **Ross McConaghy**. A creative counterpart to the bank's film-emulation B&W originals.

[Source](https://www.rossandhisjpegs.com/fujifilm-recipes/salted-slate)

| Setting | Value |
|---|---|
| Film Simulation | Acros + G filter (ACROS+G) |
| Grain Effect | Strong, Large |
| Color Chrome Effect | Off |
| Color Chrome FX Blue | Strong |
| Monochromatic Color | WC 0 & MG 0 (neutral) |
| White Balance | Auto, +3 Red & +3 Blue |
| Dynamic Range | DR100 |
| Highlight | +1.5 |
| Shadow | +4 |
| Sharpness | 0 |
| High ISO NR | −4 |
| Clarity | 0 |
| ISO | Auto (author unspecified) |
| Exposure Compensation | **−1/3 to −2/3** (underexpose for the dark slate tone) |

**Notes**
- **DR100 + Highlight +1.5 + Shadow +4** — the darkest, most crushed-shadow tone curve of any B&W in the bank, giving the heavy "slate" greys. Underexposing (−1/3 to −2/3) deepens it further and protects highlights.
- **Acros +G** — the green filter (normally for flattering skin) is used here for its mid-grey tonal rendering rather than for portraits.
- **CCFXB Strong** has no colour effect on a monochrome base, so sensor generation is irrelevant here — leave as-is.
- Sits alongside the bank's other B&W as the *creative/moody* option, vs the datasheet-faithful emulations ([Acros 100](../../originals/black-and-white/fujifilm-acros-100/), [Tri-X](../../originals/black-and-white/kodak-tri-x-400/), etc.).

> **X-Trans V — verified correct as-published (2026-07-16).** B&W (Acros) base → no colour rendering to translate; and Ross authors on an X-Trans V-category body anyway ([Pumpkin Soup](../ross-pumpkin-soup/) proof). Correct for the X-T5 as-is.
