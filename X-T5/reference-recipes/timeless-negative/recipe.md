# Timeless Negative — Fujifilm X-T5 (X-Trans V)

Nostalgic Neg. base. One of Fuji X Weekly's most popular X-Trans V recipes: a faded, print-like look with lifted shadows and rolled highlights that suits almost any subject. The name fits; frames come out looking like they could have been shot in any decade.

[Source](https://fujixweekly.com/2022/11/30/timeless-negative-fujifilm-x-t5-x-trans-v-film-simulation-recipe/)

| Setting | Value |
|---|---|
| Film Simulation | Nostalgic Neg. |
| Grain Effect | Weak, Small |
| Color Chrome Effect | Strong |
| Color Chrome FX Blue | Off |
| White Balance | Auto, +2 Red & −3 Blue |
| Dynamic Range | DR400 |
| Highlight | +2 |
| Shadow | −2 |
| Color | −3 |
| Sharpness | 0 |
| High ISO NR | −4 |
| Clarity | −2 |
| ISO | Auto, up to ISO 6400 |
| Exposure Compensation | judge per scene |

**Notes**
- The unusual tone-curve pairing (Highlight +2 with Shadow −2) is the engine of the look: punchy highlights over lifted, faded shadows, the opposite of the usual soft-highlight recipe formula.
- Color −3 pulls Nostalgic Neg.'s warmth back to a muted, printed-photo palette; Color Chrome Effect Strong keeps what colour remains from going thin.
- Clarity −2 slows shot-to-shot saving (see [grain-and-detail](../../../Knowledge/grain-and-detail.md)); set it to 0 if speed matters more than the softness.
- Native X-Trans V recipe, so no conversion is needed on the X-T5.
