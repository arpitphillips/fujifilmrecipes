# Vivid Fujichrome — a video recipe for Fujifilm cameras (X-Trans V)

Velvia base, designed for movie mode. The postcard entry in Fuji X Weekly's video-recipe series: bold, warm-leaning slide-film colour with deep blacks, for footage of colourful subjects that should look like moving Velvia frames.

[Source](https://fujixweekly.com/2026/05/02/vivid-fujichrome-a-video-recipe-for-fujifilm-cameras/)

| Setting | Value |
|---|---|
| Film Simulation | Velvia |
| White Balance | 5000K, +2 Red & 0 Blue |
| Dynamic Range | DR200 |
| Highlight | −1 |
| Shadow | −2 |
| Color | +2 |
| Sharpness | 0 |
| High ISO NR | −4 |
| Interframe NR | Auto |
| ISO | Auto, up to ISO 6400 |

**Notes**
- Shadow −2 is doing quiet work here: Velvia's native S-curve crushes shadows hard, and lifting them keeps moving footage watchable while the palette stays saturated.
- The fixed 5000K base with +2 Red bakes in a gentle golden warmth that AWB would neutralise from scene to scene.
- For stills, per the author: choose a Grain size, leave the Chrome effects Off, Clarity 0. Our own [Vivid Velvia reference](../vivid-velvia/) and [BewareMyVelvia original](../../originals/creative/bewaremyvelvia-bold/) are the stills-first cousins.
- Compatible with most X-Trans IV bodies and everything X-Trans V.
